# Fancy React Forms

This is my Learn.co React/Redux with a Rails API portfolio project.

## Overview

This app allows the creation and management of continuing education evaluations and certificates.

## Installation

Clone this repository and run

```bundle install
rails db:migrate
cd client
npm install
cd ..
rake start
```
